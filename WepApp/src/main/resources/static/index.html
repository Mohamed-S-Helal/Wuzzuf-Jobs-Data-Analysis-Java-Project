Statistical Analysis for Wuzzzuf Data Set

Main Functions:

1. Read data set and display some from it.    http://localhost:8080/sample
2. Display structure of the data set.    http://localhost:8080/structure
3. Display summary of the data set.    http://localhost:8080/summary
4. Clean the data (null, duplications) and get clean data set.    http://localhost:8080/clean
5. Count the jobs for each company and display that in order.    http://localhost:8080/top/companies
6. Show Top Demanding Companies in a pie chart.    http://localhost:8080/companies.chart
7. Get the most popular job titles.    http://localhost:8080/top/titles
8. Show the most popular job titles in bar chart.    http://localhost:8080/titles.chart
9. Get the most popular areas.    http://localhost:8080/top/areas
10. Show the most popular areas in bar chart.   http://localhost:8080/areas.chart
11. Get the most important skills required.    http://localhost:8080/skills
12. Factorize the YearsExp feature and convert it to numbers in new col.    http://localhost:8080/factorize/years
